# RickROM A05s — snapshot completo

Este repositório contém um snapshot do workspace local usado no
desenvolvimento da RickROM para o Samsung Galaxy A05s SM-A057M.

Conteúdo incluído:

- device tree;
- bengal-common;
- blobs proprietários extraídos;
- prebuilts;
- ferramentas;
- imagens e backups de partições;
- metadados Git internos arquivados.

Firmware de referência:

- Modelo: SM-A057M
- Produto: a05sub
- Plataforma: Qualcomm SM6225 / bengal
- Android: 15
- Firmware: A057MUBSCDYJ2
